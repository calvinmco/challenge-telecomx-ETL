# TelecomX - ETL Data Science Project

## Descripción

Este proyecto implementa un proceso completo de ETL (Extract, Transform, Load) utilizando Python para analizar la evasión de clientes (Churn) en la empresa TelecomX.

## Dataset

Los datos provienen de una API pública en formato JSON.

Fuente:

https://github.com/alura-cursos/challenge2-data-science-LATAM

## Tecnologías utilizadas

- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Proceso ETL

### Extract

Extracción de datos desde API en formato JSON.

### Transform

- Limpieza de datos
- Manejo de valores nulos
- Conversión de variables
- Creación de variable cuentas diarias

### Load

Análisis exploratorio:

- Distribución de churn
- Evasión por variables categóricas
- Evasión por variables numéricas
- Matriz de correlación
